Webbverktyg för vårdgivare

Allt som rör din verksamhet på ett ställe, med skräddarsydda funktioner för psykologer, fysioterapeuter & osteopater

Skapa konto eller prova direkt

Kaddio startade som ett verktyg för psykologer och klienter med en vision om att minska psykisk ohälsa

Kaddio vidareutvecklas för att både passa dig som har egen mottagning och för små till medelstora verksamheter där det finns olika discipliner, t.ex. psykologer och sjukgymnaster. För att upplevelsen ska bli bättre för både vårdtagare och vårdgivare och Kaddio ska kunna användas i hela verksamheten har vi skräddarsytt tjänsten till följande discipliner:

  • Psykologer
  • Sjukgymnaster
  • Osteopater
  • Kiropraktorer
  • Naprapater

Skräddarsydda funktioner

  Video

Träffa klienter var som helst, när som helst med video eller bara ljud

  Tidsbokning

Visa dina tillgängliga tider och låt klienter boka själva, skriv ut kvitton och dagsrapporter.

  Journalföring

Ersätt dina pärmar, papper och brandsäkra skåp med Kaddio Journalföring för psykologer

  Formulär

De vanligaste formulären för depression, impulskontroll, ångest mfl finns inbyggt. Vår 2016

  Diktering

Vid avslutad videosession kan Kaddio ge ett transkriberat förslag, du redigerar och sparar. Höst 2016

  Uppgifter

Ge uppgifter och läxor till klienter för att använda tiden mellan sessioner, spara mallar på uppgifter som kan användas igen

  Säkerhet

All data krypteras med 256-bitar AES och TLS 1.2, vi utför regelbundna tester för att hitta säkerhetsluckor. Vi tar säkerhet och integritet på allvar

  Bokmärken

Bokmärk ögonblick i videosessionen för att hitta senare, låt klienten se och höra utvalda delar när som helst. 2017
